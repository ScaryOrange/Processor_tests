# Processor_tests

Этот репозиторий демонстрирует модульное тестирование обработки векторных геопространственных данных с использованием фреймворка QGIS. Проект создан для тестирования работы с векторными слоями, фильтрации объектов, преобразования систем координат и создания буферных зон.

## Установка
Для корректного запуска требуется QGIS
### Клонирование
```
git clone https://github.com/ScaryOrange/Processor_tests project
```
В качестве интерпретатор использовать "C:\ваш_путь_к_QGIS\QGIS\bin\python-qgis.bat"
